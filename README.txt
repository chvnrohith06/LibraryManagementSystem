Steps to be followed in order to Run the Script:
1. Extract the .zip file to a desired location.
2. Start the MySQL server and make sure you have the database and proper tables created and loaded with data.
3. Once the Db connection is all set then next step is running the python main script.
4. Open the command terminal where the python script is located.
5. Type	 python3 main.py
6. The application will be launched and can perform the operations as required.

All the required Libraries and Packages need to install inorder to run the script without any error. Such as tkinter,mysql connector
Technical Dependencies to run the program:  Python 3.6.2 and MYSQL
